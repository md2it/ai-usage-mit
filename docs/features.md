## PoC Cursor

## MVP
- Предполагает, что уже есть CLI для Claude, Codex, Cursor
- Shell команда `ai-usage`
   - Опциональные флаги:
      - `-all` -- дефолтный
      - `-{provider-name}`
      - `-{model-name}`
      - Возможно несколько флагов
   - Поддерживает Claude, Codex, Cursor
   - Стандартный формат вывода для всех провайдеров и моделей
