# Reference Research Documents

This directory contains three types of documents:

- `_INDEX.yaml` — quick index of researched and planned analogs. Start here if you need a short overview, status, coverage, and comparison signals across tools.
- `{reference-id}.md` — detailed analysis of one specific tool or product. Use these files when you need evidence, documentation notes, hands-on findings, open questions, and lessons learned.
- `_flow.md` — research process. Use it only when conducting or updating research.

Recommended reading path:

- For a quick scan across tools, read `_INDEX.yaml`.
- For deep information about a specific tool, open the linked `{reference-id}.md` file from the index.
- For instructions on how to research or update these documents, read `_flow.md`.
